# 2025-26-Hockey-Networks

A repo for my data science final project that used goal data from every NHL game during the 2025-2026 season.

All coding was done through R. This is a project that will not be updated. Criticize my code all you want, I know it's inefficient, but guess what I don't care it's inefficient.

I'm not exactly sure if my goal dataset is allowed to be used because I got it is typically blocked by subscription, but the rights of use are indicated on SportsReference, so just abide by those.

The packages you need to download are either in the Shiny App itself and/or the QMD's provided.
