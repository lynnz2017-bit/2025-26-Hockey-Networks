# 2025-26-Hockey-Networks

A repo for my data science final project that used goal data from every NHL game during the 2025-2026 season to create networks for every team.

All coding was done through R. This is a project that will not be updated. Criticize my code all you want, but just know I won't care.

This is reproducible for any goal dataset pulled from SportsReference.

https://lindaenespanol.shinyapps.io/hockey_networks/ link to the Shiny App.
